Tijdlijn

Drie fundamentele technologieën voor
het World Wide Web.

1990

Dit bestond uit het idee van de Hyper Text Markup language, het URI (ook wel bekend als URL) Dit is een code die uniek is voor elk webadres. En de Hyper Text Transfer Protocol. Dit zorgde ervoor dat je naar verschillende plekken op het web kan linken. Dit samen zorgde voor de basis van het Wereld wijde web.

HTML tekst is uitgevonden

1993

Tim Berners-Lee heeft Hyper Text Markup Language tekst uitgevonden. Dit is een opmaaktaal voor webpagina’s op het World Wide Web. Het zorgt ervoor dat mensen compleet op hun eigen manier websites kunnen schrijven en deze kunnen tonen aan alle gebruikers van het WWW.

CSS is uitgevonden

1993

Cascading stylesheets zijn uitgevonden door Hakon Wium Lie. Hij zag dat het belangrijk was om ervoor te zorgen dat HTML interessant bleef en krachtiger werd door het ook visueel interessant te maken.

Het negeren van onherkenbare HTML tags

1997

Tim Berners-Lee heeft besloten dat browsers uiteindelijk niet herkenbare HTML tags moet negeren. Dit is een belangrijke uitvinding omdat dit ervoor zorgde dat er ten alle tijden nieuwe elementen gebruikt konden worden zonder dat ze moeten wachten totdat elke browser even ver is met nieuwe ontwikkelingen.

Magazine “A List Apart” brengt artikel “A Dao of Web Design” uit.

2000

John Allsopp vertelt hierin dat bijna alle nieuwe mediums voortkomen uit voorgaande mediums. Veel oude films zagen eruit als toneelstukken. De meeste oude televisieprogramma’s zagen eruit als radio gecombineerd met foto’s. Zo is het niet raar dat het begin van de webdesign vergelijkbaar was met de soorten layout van de vroegere printwereld. Zo zei John: “ “Killer Web Sites” Zijn meestal de websites die eruit zien alsof ze van papier gemaakt zijn”.

Steve Jobs ontwerpt een telefoon waarop het World Wide Web geopent kan worden.

2007

Er bestonden voor deze uitvinding al wel mobiele telefoons die dit konden, maar deze waren beperkt tot een gespecialiseerde mobielvriendelijk bestandsformaat die WML heette. Heel weinig apparaten konden HTML renderen. Door deze uitvinding vormde er zich een totaal nieuwe industrie in web design en zorgde ook voor een hoop onzekerheden.

Responsive web design is uitgevonden

2010

Mede door het opkomen van de mobiele telefoon was er een andere manier van webdesign nodig. Het ontwerp op de webpagina moest namelijk mee kunnen bewegen met de grootte van de webpagina. De technieken die dit mogelijk konden maken bestonden al. Zoals: fluid grids, flexible images, and media queries. Maar Ethan gaf deze technieken een groepsnaam namelijk; responsive design. Dit zorgde ervoor dat mensen op een andere manier na gingen denken over webdesign.

The X-ray perspective

2010

The Filament Group beschreef een techniek die ze the x-ray perspective hebben genoemd. Een x-ray perspectief aannemen betekent het kijken door de complexe widgets en visuele stijlen om zo de kale content en functionele delen die de website opmaken te identificeren. Dit is handig om te doen omdat designers vaak denken aan interacties zoals; klikken en swipen. Mensen zijn hier niet specifiek naar op zoek. Ze denken meer aan iets lezen, schrijven of kopen. Webdesigners moeten door de oppervlakte heen kijken en de belangrijke dingen daaronder zien te vinden.
